# Space out

spaceout is a simple wrapper and compression library for describing spatial and reference structures and its hash methods.

## Install

### Get npm package

`npm i --save spaceout`

or

`yarn add spaceout`

### Get Bundle.js

1. `git clone git@github.com:bagaking/space_packer.git`
2. `yarn install`
3. `yarn run build`
